#  Projeto Star Wars – Consumo de API  

Este repositório contém um projeto desenvolvido para estudos de **JavaScript**, **manipulação de DOM** e **consumo de APIs**.  
O objetivo foi criar uma aplicação simples e funcional que busca informações do universo **Star Wars** utilizando a API pública **SWAPI**.

---

##  Funcionalidades  
- Busca de personagens diretamente da API  
- Exibição de informações como nome, altura, gênero e outras características  
- Interface dinâmica montada com JavaScript  
- Tratamento de erros e respostas assíncronas  
- Layout simples e responsivo  

---

##  Tecnologias Utilizadas  
- **HTML5**  
- **CSS3**  
- **JavaScript (Fetch API)**  
- **SWAPI (https://swapi.dev/)**  

---

##  Objetivo do Trabalho  
Este projeto foi desenvolvido como parte dos meus estudos em **Desenvolvimento Web**, com foco em compreender como funciona o consumo de APIs e como integrar dados externos em páginas web.
